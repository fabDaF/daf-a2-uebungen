🔍 Entdecke Indefinitpronomen

Lies diese Sätze und achte auf die markierten Wörter:

1. Alle denken, Grammatik ist schwer, man muss nur üben und dann versteht jeder etwas.

2. Jemand hat mir gesagt, dass niemand die Prüfung bestanden hat.

3. Ich habe nichts gekauft, obwohl es einige Angebote gab.

4. Man kann in diesem Restaurant alles bestellen.

5. Manche Studenten lernen schnell, andere brauchen mehr Zeit.

6. Keiner möchte laufen. Nur einer nimmt die Treppe.

A) Nein, nie

C) Nur im Plural

Wichtige Kategorien:

Nur für Sachen: etwas, nichts

Nur für Personen: man, jemand, niemand

Für Personen und Sachen: alle, jede, einige, manche, ein, kein

📋 Grammatik-Regeln

1. etwas / nichts (nicht deklinierbar)

Nominativ Dativ Akkusativ
etwas / nichts etwas / nichts etwas / nichts

Beispiel: Ich habe nichts gekauft. Ich gebe dir etwas.

2. man / jemand / niemand

Pronomen Nominativ Dativ Akkusativ
man man einem einen
jemand jemand jemandem jemanden
niemand niemand niemandem niemanden

3. jede / manche (mit Genus)

Genus Nominativ Dativ Akkusativ
Maskulin jeder / mancher jedem / manchem jeden / manchen
Feminin jede / manche jeder / mancher jede / manche
Neutrum jedes / manches jedem / manchem jedes / manches

4. alle / einige / manche (Plural)

Pronomen Nominativ Dativ Akkusativ
alle alle allen alle
einige einige einigen einige
manche manche manchen manche

5. ein / kein (vollständig mit Plural)

Genus Nominativ Dativ Akkusativ
Maskulin einer / keiner einem / keinem einen / keinen
Feminin eine / keine einer / keiner eine / keine
Neutrum eins / keins einem / keinem eins / keins
Plural — keinen keine
